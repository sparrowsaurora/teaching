# Mary had a little lamb

using the last 3 lessons, you are to create a program that prints the following text

```txt
Mary had a little lamb,
It's fleece was white as snow;
And everywhere that Mary went
The lamb was sure to go.
```

Then you are to use variables to replace all instances of the following words:

- Mary
- lamb

so that you can set `name = "Jake"` and it will replace the name Mary with the name Jake
and you can set `animal = "Parrot"` and it will replace the animal "lamb" with the animal Parrot

## Hint:

if you want to join the variables containing words (strings), and strings (text) together,  
you can use the `+` symbol, and it will not add a space between the two lines  
and you can also use a `,`, but that will add a space  
when you join strings togewther like this, it is called "string concatenation"

```python
joined_string = "first " + variable_name, "second"
```
